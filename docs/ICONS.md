## How to Update Icons

1. Find the icon definition in `src/utils/icons.tsx`
2. Replace the SVG path data in the component
3. Test at multiple sizes (16px, 24px, 32px)
4. Ensure icon works in both light and dark modes
5. Verify icon is recognizable when small (for QR overlay)

```tsx
// Example icon component
const NewIcon = ({ className, size }: { className?: string; size?: number }) => (
  <svg 
    width={size || 24} 
    height={size || 24} 
    viewBox="0 0 24 24" 
    fill="currentColor" 
    className={className}
  >
    <path d="...SVG path data..."/>
  </svg>
)
```
